# Node Nest.js

---

## App Init

```
npm i -g @nestjs/cli
```

---

## Nest Commands

```
nest new [name]
```

```
nest generate module [name]
```

```
nest generate controller [name]
```

```
nest generate service [name]
```

```
nest generate resource [name]
```

---

## App Start

```
npm i
```

```
npm run start:dev
```
